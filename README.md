# Kokosima Reels

**Kokosima Reels** adalah repository yang berisi kumpulan eksperimen, prompt, dan struktur konten untuk pembuatan **AI-generated images & videos**, khususnya untuk format **Reels / Shorts (9:16)**.

Project ini dibuat untuk mempermudah workflow kreator AI dalam:
- Mengelola prompt
- Menjaga konsistensi visual
- Mengembangkan ide visual yang unik, absurd, atau sinematik
- Menyajikan hasil dalam format yang siap dipublikasikan ke media sosial
